Unit Circle Snake Game

Welcome to my Unit Circle Snake Game, for PC/Mac only (for now)! It plays like a normal snake game, except you can only grow by eating the correct answer to the question at the top of the screen. You have three wrong mistakes before the game is over! Use the settings to control which functions and quadrants are included. I hope this is helpful! TIP: go around in circles if you need time to think of the correct answer!
